## Automatic upgrade steps

1. Log in to the server
2. Update the latest pg node

```bash
~/node/node.sh pg update
```

3. Start pg node

```bash
~/node/node.sh pg start
```


## Manual upgrade steps

1. Download pg node

```bash
https://download.elastos.io/elastos-pg/elastos-pg-v0.0.3/
```

2. Stop pg node
3. Replace pg node
4. Start pg node

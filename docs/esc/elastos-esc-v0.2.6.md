## Automatic upgrade steps

1. Log in to the server
2. Update the latest esc node

```bash
~/node/node.sh esc update
```

3. Start esc node

```bash
~/node/node.sh esc start
```


## Manual upgrade steps

1. Download esc node

```bash
https://download.elastos.io/elastos-esc/elastos-esc-v0.2.6/
```

2. Stop esc node
3. Replace esc node
4. Start esc node

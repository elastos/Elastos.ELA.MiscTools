# The ESC v0.2.7/upgrade[20260110]

The elastos-esc-v0.2.7 version has been released. The upgrade mainly includes:

- Add and optimize blacklist functionality


## Automatic upgrade steps

1. Log in to the server
2. Update the latest esc node

```bash
~/node/node.sh esc update
```

3. Check esc status

```bash
~/node/node.sh esc status
```

4. Update the latest esc-oracle

```bash
~/node/node.sh esc-oracle update
```

5. Check esc-oracle status

```bash
~/node/node.sh esc-oracle status
```

```bash
https://download.elastos.io/elastos-esc/elastos-esc-v0.2.7/
```
